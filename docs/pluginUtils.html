CandyBarZ Plugin Utility Library Documentation BOOL CBZGetColor(HWND hwnd, long *lColor);

Description: Pops up a ColorChange Dialog and returns the RGB value of the color selected by the user.

Parameters: 
   hwnd    = (in only) handle of the calling window
   *lColor =  (in out) Address of the location to store the color.
                    The value of this parameter when it is passed in is
                    set as the initial color in the ColorChange Dialog.
Return Value: 
   TRUE  = User Pressed OK
   FALSE = User Pressed Cancel

long StringToLong(char str[]);

Parameters: 
   str    = (in only) The string to be converted

Return Value: 

   the long value of the given string
